# 📚 프로젝트 문서

이 폴더에는 Chatbot v6 프로젝트의 모든 문서가 포함되어 있습니다.

## 📄 문서 목록

### 1. SUMMARY.md
- **내용**: 초기 프로젝트 진행 상황 요약
- **작성 시점**: 프로젝트 초기
- **주요 내용**: 설정 모듈, 핵심 모듈, 전처리 모듈 구현 현황

### 2. FINAL_SUMMARY.md
- **내용**: 중간 완료 시점 요약
- **작성 시점**: 주요 모듈 완료 후
- **주요 내용**: 임베딩, 검색, 생성 모듈 구현 완료

### 3. COMPLETE_PROJECT_SUMMARY.md ⭐
- **내용**: 전체 프로젝트 완성 문서
- **작성 시점**: 모든 기능 구현 완료 후
- **주요 내용**: 
  - 전체 프로젝트 구조
  - 각 모듈 상세 설명
  - 11단계 RAG 파이프라인
  - 4가지 원칙 준수 검증
  - 확장 가능성

### 4. PERFORMANCE_REPORT.md ⭐
- **내용**: 성능 평가 보고서
- **작성 시점**: 통합 테스트 완료 후
- **주요 내용**:
  - 테스트 결과 (90.9% 성공률)
  - 성능 벤치마크 (29ms 응답, 33 QPS)
  - 모듈별 성능 분석
  - 4가지 원칙 평가
  - 개선 권장사항

## 📊 문서 읽는 순서 추천

1. **프로젝트 이해**: `../README.md` (프로젝트 루트)
2. **구조 파악**: `COMPLETE_PROJECT_SUMMARY.md`
3. **성능 확인**: `PERFORMANCE_REPORT.md`
4. **진행 과정**: `SUMMARY.md` → `FINAL_SUMMARY.md`

## 🎯 핵심 문서

프로젝트를 처음 보시는 분이라면:
- **`COMPLETE_PROJECT_SUMMARY.md`**: 전체 구조와 기능 이해
- **`PERFORMANCE_REPORT.md`**: 성능과 품질 확인
